questo è strangeOS, un OS da command line. ha un installer ISO da inserire in una vm oracle virutal box, e ha vari programmi: calcolatrice, che si entra con enterask calc esempio di calcolatrice { enterask calc utente > 10 + 9 system > [OUTPUT]= 19 utente > [QUIT] } come vedi con [QUIT] usciamo dalla calcolatrice e ritorniamo nel sistema. ci deve essere un programma semplice della gestione delle risorse, come ram usata, processore e se c'è scheda video deve avere un esplora file ci deve essere un app delle note che funziona nello stesso modo della calcolatrice solo che per salvare le note scrivi [savext (nome_nota)] questo comando salva ed esce per entrare nelle note basta scrivere enterask notes nome_nota e potremmo modificare la nota salvata nell'esplora file si devono vedere tutte le note e programmi ci deve essere il comando help per le info includi anche un gioco fantasy rpg da command line poi fai altri programmi fighi a tua scelta ci deve ancge essere un orologio e un calendario l'installazione deve esere comeun vero so, carichi l'iso, e si installa da solo. includi che nell'installazione ci sia l'italiano e l'inglese e che tu possa selezionare la data e time zone l'installazione deve essere automatica con file ISO
